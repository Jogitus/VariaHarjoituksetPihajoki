Linux GUI

Tehtävä 1

Kokeilkaa seuraavia asetuksia

1. Vaihtakaa käyttöjörjestelmän teemaa.

2. Vaihtakaa käyttöjärjestelmän taustakuvaa.

3. Vaihtakaa näytön resoluutio & virkistysaste (Hz)

Tehtävä 2

1. Luokaa Kansiosto käyttäen komentokehotetta (cmd)

2. ensin tulee päästä kotikansioomme. Käyttäen cd komentoa. Eli tulee päästä kansioon /home/OMANIMI/Documents

3. Käytä komentoa mkdir ja tee kansio "Oma Etunimi"

4. Siirry tuohon kansioon "cd oma etunimi", jonka jälkeen luo sinne neljä kansiota. Jakso1, Jakso2, Jakso3 & Jakso4.

5. Siirry pois kansioista cd .. tai cd /home/OMANIMI/Documents

6. poista tekemäsi kansio "rm -d oma etunimi" komennolla. (rm on komento remove, -d on parametri joka viittaa directoryyn ja sitten tulee directoryn nimi joka halutaan poistaa.)

Tehtävä 3

1. Aja terminalissa: "sudo apt update -y && sudo apt upgrade -y".

2. Lataa ja asenna jokin sovellus AppStoresta.

3. Asenna jokin sovellus terminalin kautta esimerkiksi: "sudo apt install cmatrix -y".

Tehtävä 4

Kokeile eri sovelluksia ja pohdi voisitko käyttää Linux Mint omassa arkielämässäsi Windowsin/MacOS sijaan.
