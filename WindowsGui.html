Windows GUI

Tehtävä 1

Kokeilkaa seuraavia asetuksia

1. Vaihtakaa käyttöjörjestelmän teemaa.

2. Vaihtakaa käyttöjärjestelmän taustakuvaa.

3. Vaihtakaa näytön resoluutio & virkistysaste (Hz)

4. Luokaa Kansiosto käyttäen komentokehotetta (cmd)

4.1 ensin tulee päästä kotikansioomme. Käyttäen cd komentoa. Eli tulee päästä kansioon /user/oppilas/

4.2 Käytä komentoa mkdir ja tee kansio "Oma Etunimi"

4.3 Siirry tuohon kansioon "cd oma etunimi", jonka jälkeen luo sinne neljä kansiota. Jakso1, Jakso2, Jakso3 & Jakso4.

4.4 Siirry pois kansioista cd .. tai cd /user/oppilas/

4.5 poista tekemäsi kansio "rm -d oma etunimi" komennolla. (rm on komento remove, -d on parametri joka viittaa directoryyn ja sitten tulee directoryn nimi joka halutaan poistaa.)

5. Aja windows update koneeseen.

6. Lataa ja asenna VirtualBox sovellus.

7. Lataa Ubuntu Desktop ISO tiedosto.

8. Lataa ja asenna Balena Etcher sovellus.

9. Luo Ubuntu Virtuaalikone Virtual Boxinn.

9.1 Anna virtuaalikoneen nimeksi "Oma nimi Ubuntu".

9.2 Valitse ISO-tiedostoksi/Levynäköistiedostoksi lataamasi ubuntu.iso

9.3 Poista ruksi "Unattended install" (Haluamme harjoitella asennusta, joten tehdään se itse eikä automaatiolla.)

9.4 Voidaan valita haluamme käyttäjätunnus. Esimerkiksi: Username - OmaNimi / Password - SalaS4na!

9.5 !HUOM! Linux koneessa ei tarvita aktivointiavainta (Product Key)

9.6 Valitaan koneen BaseMemory eli RAM. Mitä enemmän muistia, sitä nopeammin virtuaalikone toimii. Mutta suosittelen käyttämään 4096mb.

9.7 !HUOM! Virtuaalikone käyttää annetut määrät esim muistia RAMista tai Kovalevystä. Eli emme voi ylittää oman koneemme määriä. Eikä meidän kannata antaa myöskään liikaa tehoja virtuaalikoneelle.

9.8 Virtuaalikoneelle voidaan antaa "Processors", tämä tarkoittaa prosessorin coreja. Enemmän coreja virtuaalikoneelle, sitä enemmän tehoja. Suosittelen antamaan 1-2 oman koneen tehojen mukaan ja/tai virutaalikoneen käyttötarkoituksen mukaan.

9.9 Virtuaalinen kovalevyn koko kannattaa olla 20-25GB tälle ubuntu koneelle.

9.10 !HUOM! Kun valitsemme esimerkiksi 20GB virtuaalikoneen kovalevylle. Meidän omalta C: levyltä lähtee tämä 20GB varaukseen virtuaalikoneelle.

9.11 Paina asenna.
